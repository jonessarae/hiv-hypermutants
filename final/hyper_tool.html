HIV Hypermutants

This site allows the user to select a patient and find the predicted hypermutants using two tools, Hypermut and Hyperfreq. The user can also compare the results of Hypermut and Hyperfreq by selecting the "Compare Both" option. Selecting either Hypermut or Hyperfreq allows the user to see all open reading frames for each predicted hypermutant.
